Quick Viva Questions
What is Eloquent ORM?

Eloquent ORM is Laravel's built-in Object Relational Mapper that allows us to interact with database tables using models.

What does Movie::all() do?

Fetches all records from the movies table.

What does Movie::find(1) do?

Finds the record whose primary key is 1.

Why is $fillable used?

To allow mass assignment for specific columns.

Difference between Query Builder and Eloquent?
Query Builder works directly with tables.
Eloquent works through Models and is more object-oriented.

<?php

namespace Database\Seeders;

use Illuminate\Database\Seeder;
use Illuminate\Support\Facades\DB;

class MovieSeeder extends Seeder
{
    public function run(): void
    {
        DB::table('movies')->insert([
            [
                'movie_name' => 'KGF',
                'rating' => 5,
                'description' => 'Action Movie',
                'release_date' => '2022-01-10',
                'created_at' => now(),
                'updated_at' => now()
            ],

            [
                'movie_name' => 'Bahubali',
                'rating' => 5,
                'description' => 'Historical Action Movie',
                'release_date' => '2015-07-10',
                'created_at' => now(),
                'updated_at' => now()
            ],

            [
                'movie_name' => 'Pushpa',
                'rating' => 4,
                'description' => 'Crime Action Movie',
                'release_date' => '2021-12-17',
                'created_at' => now(),
                'updated_at' => now()
            ]
        ]);
    }
}
